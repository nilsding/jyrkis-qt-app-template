## Usage

1. find a good project name
2. search and replace: `yourappname` with your project name
3. you're done :)

## Requirements

- Qt 6.2.1+
- CMake (for building)

## Building

```sh
# configure it
cmake --preset dev

# compile it
cmake --build --preset dev

# run it
./build/src/yourappname
```
